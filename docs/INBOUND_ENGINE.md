# Inbound Engine

The system should not depend on manual customer hunting.

## Active offer

AI Revenue Recovery Kit

## First step

Signal Audit at 49 EUR.

## Principle

Customers should discover the gate through search, launch platforms, and public proof pages.

## Surfaces

### Search pages

Create public pages around:

- revenue recovery audit
- missed inquiry recovery
- old opportunity follow up
- sales follow up cleanup

### Public gate

Use OctoAmazonas as the public gate.

Primary call to action:

Request Signal Audit

### Proof pages

Show only safe examples:

- sample scoring categories
- sample queue format
- sample next actions
- result explanation without private data

## Conversion loop

1. Visitor finds a problem page.
2. Visitor sees the Signal Audit offer.
3. Visitor requests the audit.
4. System records the request.
5. Private commercial follow up happens after the request.
6. Delivery starts only after commercial approval.

## Rule

If a surface produces no requests, improve the headline, proof, or traffic source before adding features.
