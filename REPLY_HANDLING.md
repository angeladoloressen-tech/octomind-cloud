# Reply Handling

## Goal

Turn replies into paid audit conversations.

## Reply types

### 1. Curious

Response:

Great. The fastest way is a short audit. I map your current setup, find the runtime and memory gaps, then give you a clear cloud-first plan.

Question:

What is currently scattered: scripts, repos, API keys, local folders, automations, or deployments?

### 2. Asks price

Response:

Starter audit is 490 EUR and takes 48 hours. You get a system map, runtime path, safety checklist, and next-action plan.

### 3. Wants example

Response:

Example result: local scripts and unfinished automations become a cloud runtime, GitHub taskboard, memory layer, and safety gate.

### 4. Not now

Response:

No problem. I can send a short checklist you can use to see if your AI setup is getting too scattered.

## Rule

Do not argue. Qualify, clarify, then offer the audit.
