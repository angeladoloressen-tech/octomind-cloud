# Lead Scoring

## Purpose

Prioritize leads for the Cloud Command Center Audit.

## Hot lead signals

- has AI scripts or agents
- has local automations
- uses many API keys
- has unfinished prototypes
- has unclear deployment
- has client work or budget
- asks about cleanup or system design

## Score

Hot: 60+
Warm: 30-59
Cold: below 30

## First question

What is currently scattered: scripts, repos, API keys, local folders, automations, or deployments?

## Action

Hot leads get a personalized audit offer.
Warm leads get a useful checklist.
Cold leads are not chased.
