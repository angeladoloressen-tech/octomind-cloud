# Enterprise Discovery Questions

## Goal

Diagnose whether a prospect needs AI Operations Reset or a lighter audit.

## Questions

1. Which tools currently run your client or internal workflows?
2. Where do automations break or become hard to monitor?
3. Who approves workflow changes before they affect clients or customers?
4. Where are API keys, docs, owners and deployment steps tracked?
5. Do you have a memory or knowledge layer for automation decisions?
6. Which parts of delivery still depend on one person remembering details?
7. Where is AI being used today?
8. What would be costly if automated incorrectly?
9. What would improve if workflows had a command layer?
10. Is this a one-time cleanup or an ongoing operating layer?

## Qualification

If the prospect has client delivery, multiple tools, unclear approvals and AI interest, offer AI Operations Reset.

If they are early or small, offer the 490 EUR diagnostic.
