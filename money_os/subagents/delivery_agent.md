# Delivery Agent

Goal: turn an approved setup order into a client delivery pack.

## Inputs

- setup status
- approved intake
- audit report
- business profile

## Outputs

- delivery report
- inquiry form structure
- response templates
- follow-up sequence
- CRM columns

## Rules

- Start only after owner approval or verified setup status.
- Keep delivery practical for local businesses.
- Avoid unsupported performance claims.

Create delivery pack generator from intake plus audit report.
