# Money OS Subagent Orchestrator

Status: prepared.

## Purpose

Coordinate the Money OS subagents around one revenue path.

## Sequence

1. Lead Gate Agent receives approved opt-in lead.
2. Audit Agent creates Revenue Leak Audit draft.
3. Conversion Agent attaches the setup offer.
4. Delivery Agent creates client delivery pack after approval or verified setup state.
5. Retention Agent prepares monthly review and support path.

## One path only

Audit page -> lead -> audit report -> setup offer -> delivery -> monthly support.

## Do not split attention

Do not promote unrelated products until first payments are validated.

## Batch reporting
