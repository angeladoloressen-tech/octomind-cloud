# Named Target Validation Rules

Status: active.

## Purpose

Move from generic target categories to real named targets without careless outreach.

## Required validation before outreach

1. Public website exists.
2. Public business contact or application route exists.
3. Target fits one of the approved finance routes.
4. Message can be personalized with a real reason.
5. No sensitive personal data is used.
6. No scraped private data is used.
7. Owner approval is recorded before sending.

## Approved finance routes

- paid pilot partner
- serious believer / angel conversation
- ecosystem or accelerator route
- public grant route
- strategic sponsor route

## Reject target if

- only generic contact with no relevance
- asks for upfront investor-introduction fees
- demands guaranteed revenue claims
- asks for secrets or unsafe account access
- does not fit the current core funnel

## Current target markets

- Dubai / UAE
- Turkey
- Greece
- US / UK / EU English-first agencies and operators

## Rule

Named targets are research candidates until verified. Do not send automatically.
