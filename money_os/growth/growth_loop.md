# Growth Loop

Purpose: help Money OS improve through small measured experiments.

## Loop

1. Observe.
2. Pick one hypothesis.
3. Build one small test.
4. Measure one result.
5. Keep, improve, or stop.
6. Save the learning.
7. Start the next batch.

## Current focus

Improve the Revenue Leak Audit funnel.

## Rule

Prefer improving the current funnel before adding new offers.
