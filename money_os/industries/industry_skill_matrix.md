# Industry Skill Matrix

Status: prepared, not launched.

## Purpose

Map each target industry to the skill modules needed for the same core funnel.

## Core funnel

Revenue Leak Audit -> AI Revenue Autopilot Setup -> Monthly Support.

## Matrix

| Industry | Highest pain | Required skill modules | First channel |
|---|---|---|---|
| Beauty studios | missed bookings | lead capture, audit diagnosis, traffic asset, retention | Instagram / local profile |
| Driving schools | slow inquiry handling | lead capture, audit diagnosis, offer clarity, delivery pack | Google profile / local search |
| Cleaning companies | quote chaos | lead capture, offer clarity, payment ops, delivery pack | Kleinanzeigen / Google profile |
| Physio and wellness | appointment gaps | lead capture, retention, delivery pack, cashflow cockpit | local search / Instagram |
| Small B2B agencies | scattered AI ops | audit diagnosis, offer clarity, cashflow cockpit, compliance gate | LinkedIn permission-based |
| Local trades | missed calls and quotes | lead capture, delivery pack, payment ops, traffic asset | Kleinanzeigen / local referrals |

## Rule

Pick only one industry per experiment. Do not run all at once.

## Emergency recommendation

Start with either beauty studios or small B2B agencies because existing assets already point in those directions.
