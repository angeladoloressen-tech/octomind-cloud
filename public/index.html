PUBLICATION BOUNDARY

GitHub is the observatory, not the private brain.

Only sanitized public exports belong here. Private source code, credentials, customer data, raw internal state, and confidential evidence remain outside the public portal.

Allowed

Public status summaries, release notes, governance decisions, non-sensitive evidence, and reproducible demonstrations.

Blocked

Secrets, API keys, private logs, personal data, customer records, payment details, and unreviewed autonomous actions.

Required

Human approval, provenance, policy classification, clear limitations, and a rollback path for every public release.
