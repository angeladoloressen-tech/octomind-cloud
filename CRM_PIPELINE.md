## Lead fields

- name
- channel
- fit reason
- current chaos
- next action
- status
- notes

## Qualification checklist

Good signs:

- has many AI tools or scripts
- no clear deployment path
- uses local machine as runtime
- has many unfinished prototypes
- needs safety or cleanup
- can pay for speed and clarity

Bad signs:

- wants risky automation
- wants free consulting
- cannot describe any real project
- wants guaranteed income without work

## First action

Ask one question:

What is currently scattered in your system: scripts, repos, API keys, local folders, automations, or deployments?
